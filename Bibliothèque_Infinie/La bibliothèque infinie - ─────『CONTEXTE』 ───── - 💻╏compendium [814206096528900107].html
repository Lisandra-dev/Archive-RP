Le compendium est un site internet bien connu des invisibles. Il permet de rechercher rapidement, dans une base de données régulièrement mise à jour, les objets magiques qui ont été recensé. Les objets magiques peuvent provenir : ‚ñ´Ô∏è De fouille archéologique ‚ñ´Ô∏è De fabrication (par diverses entreprises) ‚ñ´Ô∏è De découverte impromptue De nombreuses filières travaillent sur les objets magiques, et cette base de données est la plus complète au monde. Les objets présent dans la base de données sont de natures diverses : elles peuvent être des armes, mais aussi des livres, des objets simples voire électroniques. La présentation des objets est plutôt simple :
Nom de l'objet Type : Description : Effet : (image éventuelle)
Un onglet est présent sur le compendium afin de relier les recherches qui ont été faites dessus, ainsi que d'autres objets qui pourraient être relié à lui. Tout le monde peut rajouter un objet, mais l'ajout sur la base de donnée nécessite la validation par un ou plusieurs modérateur (issus généralement de faculté reconnue, ou étant des professeur reconnu dans leur matière), qui se déplaceront sur le lieu pour vérifier si l'objet n'est pas un doublon, et s'il est réellement magique.
