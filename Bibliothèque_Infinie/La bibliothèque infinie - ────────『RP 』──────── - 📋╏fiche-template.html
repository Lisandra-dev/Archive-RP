Si vous ne trouvez pas d'image qui vous conviennent, voici une liste de picrew sympathique : https://raindrop.io/collection/17535279 Si vous avez des questions, posez les dans #🤔╏questions. Vous pouvez aussi créer un ticket pour les questions qui concernent des données sensibles sur votre personnage.
Les fiches PNJ utilise la même template. Cependant, il est autorisé à faire des fiches moins fournie que pour un personnage.
Il n'y a pas de maximum de personnage autorisé, mais pour des raisons de Fairplay, les combats doivent avoir le même nombre de combattant s'il y a du multi-personnage.
Pour faciliter la correction, merci de limiter le nombre de mots de votre fiche à maximum 4000 mots
Toutes les fiches sont disponibles à la consultation. Si vous ne souhaitez pas que des infos soient connus des joueurs (histoire, personnalité), ne les mettez pas. Vous devez tout de même décrire un maximum vos pouvoirs et capacités. Le recensement se trouve ici : https://drive.google.com/drive/folders/1zDs3L7v52EBR3okoh7E8LYed4IsBr9Kq?usp=sharing (edited)
